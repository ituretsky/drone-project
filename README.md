# Drone Programming Portfolio

## О проекте
Этот репозиторий содержит учебные проекты по программированию автономных дронов на базе **ArduPilot SITL**, **MAVProxy** и **DroneKit-Python**. Все скрипты написаны на Python 3.10 и протестированы в симуляторе.

---

## Проекты

### 1. Первый полёт (`first_flight.py`)
- **Задача:** взлёт на 5 метров и посадка
- **Что отрабатывается:** подключение к симулятору, переключение режима GUIDED, работа с `simple_takeoff()` и `LAND`
- **Результат:** дрон успешно взлетает и садится по команде

### 2. Полёт по квадрату с логированием (`square_flight_w_log.py`) — актуальный скрипт
- **Задача:** пролететь по квадрату со стороной 100 метров с записью телеметрии
- **Что отрабатывается:** пересчёт метров в градусы, движение по точкам (`simple_goto`), логирование в CSV
- **Среда:** ArduPilot SITL + MAVProxy (без физического дрона)
- **Результат:** дрон выполняет маршрут, данные полёта сохраняются в `track.csv`

---

## Инструменты и стек
- **ArduPilot SITL** — симулятор автопилота
- **MAVProxy** — шлюз для MAVLink-команд
- **DroneKit-Python** — библиотека для управления дроном
- **Python 3.10** — язык разработки
- **Git / GitHub** — контроль версий
