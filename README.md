# Drone Programming Portfolio

Проекты по программированию автономных дронов на базе **ArduPilot SITL**, **DroneKit** и **ROS/Gazebo** (в перспективе).

---

## 🔧 Инструменты и стек

- **Python 3.10** — основной язык разработки.
- **DroneKit** — библиотека для управления дроном из Python.
- **MAVProxy** — MAVLink-шлюз для связи с симулятором.
- **ArduPilot SITL** — симулятор автопилота.
- **Matplotlib** — визуализация полётных логов.
- **Git** — контроль версий.

---

## 📁 Проекты

### 1. Планировщик миссий с мониторингом  
**Файл:** `mission_planner_mon.py`

**Возможности:**
- Взлёт, полёт по квадрату/прямоугольнику, посадка.
- Чтение параметров миссии из JSON-конфига.
- Логирование полёта в CSV (время, координаты, высота, режим).
- Фоновый мониторинг телеметрии в реальном времени.

**Запуск:**

```bash
# Терминал 1 — симулятор (вне виртуального окружения)
cd ~/ardupilot
sim_vehicle.py -v ArduCopter --map --console

# Терминал 2 — скрипт (внутри виртуального окружения)
cd ~/drone_project
source drone_env/bin/activate
python3 mission_planner_mon.py
```

---

### 2. Анализатор полётных логов  
**Файл:** `telemetry_analyzer.py`

Читает CSV-лог и строит графики:
- **Траектория полёта** (широта/долгота).
- **Высота от времени**.
- **Режимы полёта** (GUIDED, LAND и т.д.).

**Результат:** PNG-файлы с графиками.

**Запуск:**

```bash
cd ~/drone_project
source drone_env/bin/activate
python3 telemetry_analyzer.py
```
